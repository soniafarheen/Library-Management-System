# 📚 Library Management System with Admin Dashboard

### 🎓 2nd Semester OOP Project | Java Swing + MySQL

---

## 📖 Project Overview

A full-featured **Library Management System** with an **Admin Dashboard**, built from scratch as part of my **Complex Engineering Problem (CEP)** assignment in 2nd semester.  

The system streamlines library operations with secure admin access, allowing authorized personnel to manage students, books, and transactions efficiently.

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🔐 **Secure Admin Login** | Authorized access only |
| 👥 **Student Management** | Full CRUD operations for student records |
| 📚 **Book Management** | Add, update, delete, and search books |
| 📤 **Issue Book** | Issue books to students with tracking |
| 📥 **Return Book** | Process returns and update availability |
| 📊 **View Statistics** | Monitor book activity and system usage |
| 🎨 **Custom GUI** | Drag-and-drop designed interface with custom icons |

## 🚀 How to Run

1. Clone this repository  
2. Open project in **NetBeans IDE**  
3. Set up **MySQL database** (import SQL file if available)  
4. Configure database connection in the code  
5. Run the application
